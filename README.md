# ⛏️ Minecraft Themed Planner

## 📖 Overview
An all-in-one productivity dashboard featuring a scheduler, priority task board, and focus timer. This project experiments with UI gamification, with a Minecraft Theme to make daily organization more engaging.

## 🛠️ Focus Areas
* **Thematic UI/UX:** Strict adherence to visual branding using Tailwind CSS.
* **State Management:** Handling complex interactions between timers and task lists.
* **User Engagement:** Applying game design logic to standard business tools.
